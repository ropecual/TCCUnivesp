O tempo real de execução das trilhas foi obtido automaticamente a partir da diferença entre os timestamps inicial e final registrados nos arquivos GPX, evitando dependência de tempos declarados manualmente pelos praticantes.


A distância total da trilha foi calculada por meio da soma das distâncias geodésicas entre pontos consecutivos do arquivo GPX, enquanto o ganho de elevação positivo foi obtido a partir da soma das variações altimétricas ascendentes
